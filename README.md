# AdrenalineROM Custom Menu

### Current Functions
1. Remap Bixby Key
2. Change Device Name
3. Max Volume Warning Toggle
4. Navigation Bar Toggle
5. Immersive Mode
6. Set DNS
7. [Powercfg](https://github.com/yc9559/cpufreq-interactive-opt "Powercfg") by yc9559 @ CoolApk **[EXY8895 ONLY]**

### Usage
`su`
`cfg`
